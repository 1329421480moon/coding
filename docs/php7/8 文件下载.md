## 文件下载

```
<?php
$file="houdunwang.jpg";

//指定下载文件类型为二进制
header("Content-type:application/octet-stream");

//获取文件名
$fileName = basename($file);

//下载窗口显示文件名
header("Content-Disposition:attachment;filename={$fileName}");

//文件尺寸单位
header("Accept-ranges:bytes");

//文件大小
header("Accept-length:".filesize($file));

//读取文件内容供下载
readfile($file);
```
